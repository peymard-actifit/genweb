# Genweb

Projet Genweb - Générateur de sites web.

## Description

Ce projet est en cours de développement.

## Branches

- `main` - Branche principale de production
- `features` - Développement de nouvelles fonctionnalités
- `fixes` - Corrections de bugs
- `hotfixes` - Corrections urgentes en production

## Installation

```bash
# Cloner le repository
git clone https://github.com/peymard-actifit/genweb.git
```
